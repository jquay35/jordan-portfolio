# Jordan Quay Portfolio

This is my personal portfolio website created for my React final project. The website showcases my background, creative interests, featured projects, and contact information.

## Project Description

This portfolio was built using React and Vite. It includes multiple pages with React Router, a consistent visual theme, responsive design, project cards, images, and a contact form with validation.

## Features

- Home page with hero image
- About page with bio, skills, and image
- Projects page with three featured projects
- Contact page with form validation
- Navbar and footer on every page
- Responsive layout for desktop and mobile
- Styled with CSS
- Built with React Router

## Technologies Used

- React
- Vite
- React Router DOM
- HTML
- CSS
- JavaScript
